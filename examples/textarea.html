With Character Counter

0 / 200 characters 

<textarea maxlength="200" oninput="updateCounter(this, 'counter-id')"></textarea>
<p class="text-sm text-muted-foreground">
  <span id="counter-id">0</span> / 200 characters
</p>

<script>
function updateCounter(textarea, counterId) {
  document.getElementById(counterId).textContent = textarea.value.length;
}
</script>

Auto-resize (JavaScript)

This textarea automatically adjusts its height as you type. 

<textarea class="... resize-none overflow-hidden" oninput="autoResize(this)"></textarea>

<script>
function autoResize(textarea) {
  textarea.style.height = 'auto';
  textarea.style.height = textarea.scrollHeight + 'px';
}
</script>

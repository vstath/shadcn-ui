Accessible Dialog Component

Fully accessible modal dialog with ARIA attributes, keyboard navigation, focus management, and CSP compliance.

Accessibility Features

  • Focus trap: Tab cycles through dialog elements only
  • Escape key: Closes dialog and restores focus
  • Focus restoration: Returns focus to trigger button on close
  • ARIA attributes: role="dialog", aria-modal="true", aria-labelledby, aria-describedby
  • Body scroll lock: Prevents background scrolling when dialog is open
  • Screen reader announcements: Dialog open/close announced
  • CSP compliant: No inline event handlers

HTML Structure

<!-- Trigger Button -->
<button data-dialog-trigger="my-dialog">
  Open Dialog
</button>

<!-- Dialog -->
<div
  id="my-dialog"
  role="dialog"
  aria-modal="true"
  aria-labelledby="my-dialog-title"
  aria-describedby="my-dialog-description"
  aria-hidden="true">

  <!-- Overlay -->
  <div data-dialog-overlay></div>

  <!-- Content -->
  <div data-dialog-content class="...">
    <h2 id="my-dialog-title">Dialog Title</h2>
    <p id="my-dialog-description">Dialog description</p>

    <!-- Your content -->

    <button data-dialog-close>Close</button>
  </div>
</div>

import { initDialogs } from '../src/js/components/dialog.js';
